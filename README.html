Contenido del repositorio:

Los scripts contenidos en este repositorio miden los indicadores de los atributos del derecho a cuidar y del derecho a ser cuidada o cuidado, de acuerdo con la metodologa desarrollada en el documento Marco analtico y metodologa para diagnosticar las brechas en los cuidados en municipios de Mxico.

Para comprender lo que se mide en cada indicador es necesario antes revisar documento mencionado.

Scripts:

Los scripts estn agrupados siguiendo la lgica del documento, de tal manera que la carpeta Derecho a ser cuidadx - Cobertura contiene todos los scripts que miden cobertura de los distintos cuidados (personal, de desarrollo integral, de salud) para los tres grupos de inters (infancia temprana, personas con discapacidad, personas adultas mayores). Ntese que el Derecho a cuidar no se divide por tipo de cuidado, por lo que todos los scripts correspondientes a este derecho se alojan dentro de la nica carpeta Derecho a cuidar.

El nombre de los scripts corresponde a la siguiente informacin: ### Nombres de los scripts

\[Grupo de inters + _ + Tipo de cuidado + _ + Atributo del derecho + _ + Proveedor del servicio\]

Por ejemplo, IT_CP_Cob_Hog.R es el script que mide el indicador de cobertura de cuidados personales para infancia temprana desde el hogar, y as sucesivamente.

El nico script que no cabe dentro de sta lgica es A1_EstimacionesCrecimientoPoblacional.r que corresponde a la informacin del Anexo 1 del documento metodolgico.

Replicabilidad con otros municipios

Cada script est hecho de tal manera que el diagnstico que se realiz para Manzanillo, Colima se puede recrear para cualquier municipio del pas.

Para ello, cada script tiene una seccin llamada Seleccionar municipio y estado de inters (o, en sus variaciones, seleccionar municipio de inters o seleccionar estado de inters segn la representatividad de las bases utilizadas para ese indicador) en la que se le pide a la persona usuaria revisar el cdigo correspondiente al municipio y estado de inters (en una base del INEGI) y cambiar el cdigo en los objetos sealados. Despus de este cambio, el script debe correr sin contratiempos para la localidad seleccionada.

Versin de R utilizada

Todos los scripts se corren en la versin 4.0.3 de R, desde RStudio.

Sobre el sistema operativo

Todos los scripts se pueden correr desde Mac o PC.
